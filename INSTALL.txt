🖥️ ПК ПРИЛОЖЕНИЕ (Windows)
═══════════════════════════════════════════════════════════

Автоматическая установка:
1. Запусти: build_pc_app.bat
2. Подожди 3-5 минут
3. На рабочем столе появится "Daily Rise.exe"
4. Запусти приложение!

Требования:
• Node.js 18+ (https://nodejs.org)
• Python 3.10+ с установленными зависимостями


📱 ANDROID ПРИЛОЖЕНИЕ (APK)
═══════════════════════════════════════════════════════════

Способ 1 - Онлайн (просто):
1. Открой https://pwabuilder.com
2. Введи URL: http://твой-ip:8000
3. Нажми "Build for Android"
4. Скачай и установи APK

Способ 2 - Локально (продвинуто):
1. Установи: pip install kivy buildozer
2. Запусти: build_apk_simple.bat
3. Выбери "Y"
4. APK будет в папке bin/


🌐 ВЕБ ВЕРСИЯ (браузер)
═══════════════════════════════════════════════════════════

1. Запусти: run_app.bat
2. Открой: http://localhost:8000
3. Для доступа с телефона:
   - Узнай IP: ipconfig
   - Открой с телефона: http://твой-ip:8000

❓ ПРОБЛЕМЫ
═══════════════════════════════════════════════════════════

"Node.js не найден"
→ Установи с https://nodejs.org (LTS версия)

"Python не найден"
→ Установи с https://python.org (3.10-3.12)

"ModuleNotFoundError"
→ Запусти: pip install -r requirements.txt

"Не открывается с телефона"
• Проверь что ПК и телефон в одной WiFi сети
• Открой брандмауэр для порта 8000
• Используй IPv4 адрес (ipconfig)
