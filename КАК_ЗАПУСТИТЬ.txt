  Daily Rise - Как запустить
═══════════════════════════════════════════

📁 Папка с приложением:
   C:\Users\Илья\habit_tracker

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ СПОСОБ 1 - Через файл (просто)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Открой папку:
   C:\Users\Илья\habit_tracker

2. Дважды кликни на файл:
   📄 start_app.bat

3. Приложение откроется в браузере!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🖥️ СПОСОБ 2 - Ярлык на рабочий стол
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Открой папку:
   C:\Users\Илья\habit_tracker

2. Найди файл: start.vbs

3. Перетащи его на рабочий стол
   (или скопируй → вставь на рабочем столе)

4. Готово! Теперь можно запускать 
   с рабочего стола.

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📱 ДОСТУП С ТЕЛЕФОНА
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Запусти приложение (способ 1)

2. Узнай IP компьютера:
   - Нажми Win+R
   - Введи: cmd
   - Введи: ipconfig
   - Найди: IPv4-адрес (например, 192.168.0.10)

3. Открой с телефона:
   http://твой-ip:8000

→ Установи Python 3.10+ с python.org

"venv не найдено"
→ Запусти install.bat

"Ошибка подключения"
→ Подожди 5 секунд после запуска
