  🌱 Daily Rise - Полная инструкция
═══════════════════════════════════════════════════════════

📦 ЧТО В ПАПКЕ:
═══════════════════════════════════════════════════════════

🖥️ Для компьютера:
   • make_desktop_app.bat - Создать EXE приложение
   • desktop.py - Исходный код приложения
   • dist/ - Сюда соберётся приложение

📱 Для телефона:
   • app/static/manifest.json - PWA манифест
   • ANDROID_APK.md - Инструкция по созданию APK

🔧 Скрипты:
   • install.bat - Установить зависимости
   • start_app.bat - Запустить в браузере
   • setup.bat - Быстрая установка


═══════════════════════════════════════════════════════════
  🖥️ DESKTOP ПРИЛОЖЕНИЕ (ОКНО НА ПК)
═══════════════════════════════════════════════════════════

1. Запусти:
   make_desktop_app.bat

2. Подожди сборку (~2-5 минут)

3. После сборки:
   • dist/Daily Rise.exe - приложение
   • Ярлык на рабочем столе

4. Запусти приложение!

⚠️ Важно:
   • Приложение включает встроенный сервер
   • Работает без браузера (отдельное окно)
   • Не закрывай окно пока пользуешься


═══════════════════════════════════════════════════════════
  📱 ANDROID ПРИЛОЖЕНИЕ (APK)
═══════════════════════════════════════════════════════════

📌 СПОСОБ 1 (простой - PWA):

1. Запусти приложение на ПК

2. Открой с телефона:
   http://192.168.x.x:8000
   (IP компьютера узнай через ipconfig)

3. Нажми 📲 в шапке

4. Следуй инструкции

5. Иконка появится на главном экране!


📌 СПОСОБ 2 (онлайн - APK):

1. Нужен HTTPS (например, через ngrok)

2. Открой https://pwabuilder.com

3. Введи свой URL

4. Скачай APK

5. Установи на телефон


📌 СПОСОБ 3 (локально):

1. Смотри ANDROID_APK.md

2. Нужны Java JDK + Android SDK

3. Сборка через Bubblewrap


═══════════════════════════════════════════════════════════
  ❓ ПРОБЛЕМЫ
═══════════════════════════════════════════════════════════

"Python не найден"
→ Установи Python 3.10+ с python.org

"ModuleNotFoundError"
→ Запусти install.bat

"Не открывается окно"
→ Подожди 10 секунд после запуска

"Не работает с телефона"
• Проверь что ПК и телефон в одной WiFi сети
• Открой брандмауэр для порта 8000
• Используй IP компьютера (не localhost)
